## ✨ Funcionalidades

- 🌡️ **Clima em tempo real** — temperatura, sensação térmica, umidade, vento e pressão
- 📰 **Notícias da cidade** — artigos recentes buscados automaticamente
- 🌍 **Qualquer cidade do mundo** — basta digitar o nome
- 🎨 **Design dark moderno** — interface clean com visual de dashboard
- 📱 **Responsivo** — funciona em mobile e desktop
- ⚡ **Sem frameworks** — HTML, CSS e JavaScript puro

## 🚀 Como rodar

### 1. Clone o repositório
```bash
git clone https://github.com/gabriellrodriguess/weatherpulse.git
cd weatherpulse
```

### 2. Configure as chaves de API

No arquivo `index.html`, localize as linhas:

```javascript
const WEATHER_API_KEY = 'SUA_CHAVE_OPENWEATHERMAP';
const NEWS_API_KEY    = 'SUA_CHAVE_NEWSAPI';
```

Substitua pelos valores das suas chaves:

- 🔑 **OpenWeatherMap** → [openweathermap.org/api](https://openweathermap.org/api) *(gratuito)*
- 🔑 **NewsAPI** → [newsapi.org](https://newsapi.org) *(gratuito)*

### 3. Abra no navegador

Basta abrir o `index.html` direto no navegador. Nenhuma instalação necessária.
